# Real-time Chat App + Live Code Editor

A full stack real-time chat application with an integrated collaborative code editor.

Built using React.js, Node.js, Express.js, Socket.io and MongoDB.


## Features
### Real time chat
- one to one real-time messaging using socket.io
- send text, images, videos
- online/offline user presence

### Real time code collaboration
- create or join rooms via Room ID
- Live collaborative code editing
- Automatic sync , share Room ID with other users to collaborate

### User Authentication
- Email + password login/registration
  
## Tech Stack
- React.js, Node.js, Express.js, MongoDB, socket.io
